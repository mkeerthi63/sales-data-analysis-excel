# Sales Data Analysis Dashboard (Excel)

## Project Overview

This project analyzes retail sales data using **Microsoft Excel** to identify key business insights.
The goal is to transform raw sales data into a **visual and interactive dashboard** that helps understand sales trends, product performance, and regional performance.

The dashboard allows users to filter and explore the data using **interactive slicers**, making it easier to analyze different regions and product categories.


## Dataset Description

The dataset contains sales transaction records with the following fields:

| Column       | Description                             |
| ------------ | --------------------------------------- |
| Order_ID     | Unique order identifier                 |
| Order_Date   | Date when the order was placed          |
| Region       | Sales region (North, South, East, West) |
| Category     | Product category                        |
| Product_Name | Name of the product                     |
| Quantity     | Number of units sold                    |
| Sales        | Total sales amount                      |
| Profit       | Profit generated from the sale          |

## Project Steps

### 1. Data Cleaning

* Checked for missing values
* Ensured correct data types
* Converted the dataset into an **Excel Table**

### 2. Data Analysis using Pivot Tables

Created multiple Pivot Tables to analyze:

* Total Sales by Region
* Sales by Product Category
* Profit by Category
* Quantity sold by Product

### 3. Data Visualization

Generated charts from Pivot Tables:

* Bar Chart → Sales by Region
* Pie Chart → Sales by Category
* Column Chart → Profit by Category
* Bar Chart → Top Selling Products

### 4. Dashboard Creation

A dashboard sheet was created to display:

**Key Performance Indicators (KPIs)**

* Total Sales
* Total Profit
* Total Quantity Sold

**Charts**

* Sales by Region
* Sales by Category
* Profit by Category
* Top Products

### 5. Interactive Filters

Added **Slicers** for:

* Region
* Category

These slicers allow users to dynamically filter and explore the dashboard.


## Key Insights

* The **North region generated the highest sales revenue**.
* **Electronics category produced the highest sales**.
* **Laptop and Mobile were among the top-selling products**.
* Furniture products produced strong profit margins.
